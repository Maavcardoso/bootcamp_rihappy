Tipografia HTML

  • p - Paragráfo
  • blockquote - Citação

  • mark - Marcador de texto

Listas

Lista Ordenadas (ol)

  1. É ordenada em números
  2. A tag para cada item da lista é li (list item)
    1. Também é possível inserir uma lista ordenada de forma aninhada

Lista Não ordenada (ul)

  • Funciona da mesma forma que uma lista ordenada, contudo, sem numeração
  • Listas são uma forma interessante de se juntar diversas informações em uma só tag
  • Sendo utilizada especialmente em menus.

Links (a)

Para inserir um link na página, deve se inserir <a href="link aqui">, sendo href o link que referencia a página. Há também dois atributos possíveis: title="", para alterar o label do link, e target="", para designar se o link será aberto na mesma aba ou em uma nova aba.

CLIQUE AQUI

Para o atributo target, há duas opções:

  • target="_self" - Para abrir o link na aba atual
  • target="_blank" - Para abrir o link em uma nova aba
