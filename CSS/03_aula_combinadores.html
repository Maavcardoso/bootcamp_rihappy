Agrupamento de Seletores

Podemos aplicar as mesmas regras CSS para vários seletores diferentes. Para isso, podemos separar os seletores utilizando uma virgula.
Assim, não precisamos copiar e colar o código para todos, basta escrever um bloco de CSS e reutilizar as mesmas regras!

Combinadores

Combinadores são o tipo de relação entre dois 2 seletores, seguindo a ordem:
[SELETOR] [COMBINADOR] [SELETOR]

  1. Combinador descendente (espaço);
  2. Combinador filho (>);
  3. Combinador irmão adjacente (+);
  4. Combinador irmão em geral (-);

Combinador Descendente

Combina dois seletores para que os elementos correspondidos pelo segundo seletor sejam selecionados se eles tiverem um elemento ancestral correspondente ao primeiro seletor.
