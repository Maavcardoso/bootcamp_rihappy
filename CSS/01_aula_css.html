INTRODUÇÃO AO CSS

Formas de declaração

  • Inline

    O código CSS é adicionado com o atributo <style> diretamente na tag HTML, um a um. Tem prioridade sobre outras formas de declaração.
    Exemplo:

    CSS INLINE

    <p style="background-color: #0000ff; color:#ffffff;font-size: 16px;font-weight: bolder;">CSS INLINE</p>

  • Interno

    É adicionado dentro da tag head da página HTML. Dentro de head, adicionamos a tag style e adicionamos o CSS nesta área.

    CSS INTERNO

    Inspecione o elemento head com a tecla F12 e veja o código do estilo acima.

  • Externo

    É criado um arquivo com a extensão .css com todas as regras CSS que queremos aplicar. O mesmo estilo pode ser utilizado em diferentes paginas.
    Esse arquivo é referenciado no HTML da página através da tag link dentro da tag head.

    CSS EXTERNO

    <link rel="stylesheet" href="./assets/css/01aula.css">
