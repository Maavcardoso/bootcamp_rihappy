SELETORES CSS

  • Seletor por Tipo: Busca elementos por uma tag HTML especifica

  • Seletor por ID (#): Busca elementos através do atributo id. Uma ID pertence a apenas um elemento.

  • Seletor por Classe (.): Busca elementos através do atributo class. Um elemento pode ter mais de uma classe, separando cada classe com espaço no atributo "class" e uma classe pode ser utilizada em vários elementos.

  • Seletor Universal (*): Seleciona todos os elementos HTML. Por exemplo, todos os elementos nesta página estão em negrito.

  • Seletor de atributo ([atrib]): Seleciona elementos que possuem um atributo especifico em nosso documento HTML e conseguimos também buscar atributos com um valor específico.

      Operadores de busca

    • [attr = value]: Seleciona o elemento com nome attr que contenha o valor exatamente value.
      Youtube

    • [attr ~= value]: Seleciona o elemento com atributo de nome attr, o qual value é uma lista de palavras separadas por espaços, e uma dessas é exatamente value.
      Wikipedia

    • [attr |= value]: Seleciona o elemento com atributo de nome attr, o qual o valor pode ser exatamente value ou pode começar com value imediatamente seguido por hífen.
      Google

    • [attr ^= value]: Seleciona o elemento com o atributo de nome attr, que tem como valor prefixado por value (começa com value).
      Github

    • [attr $= value]: Seleciona o elemento com o atributo de nome attr, que tem como sufixo por value (termina com value).

    • [attr *= value]: Seleciona o elemento com atributo de nome attr, o qual valor contém ao menos uma ocorrência de value.
