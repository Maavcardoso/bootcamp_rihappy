SEMÂNTICA HTML

O que significa semântica?

O texto em html é formatado através das tags. Além da formatação visual, as tags também carregam um sentido que, na hora de ser interpretado pelos motores de busca e leitura, ganham relevância e destaque.
Por exemplo, qual a diferença entre strong e bold?
Apesar de os dois estarem em negrito, o motor de busca assimilará a relevância da informação de forma distinta.

Tag font

Esta tag é utilizada para formatar letras. O CSS apresenta mais modularidade e possiblidades para uso de fontes, mas entender o html ajuda a compreender os futuros atributos passados por css.

Este é um exemplo

<h2><font color="#ee0000" face="'Trebuchet MS',Tahoma"> Este </font>é um exemplo</h2>

  • atributo color: É possível selecionar uma cor através de código RGB, CMYK e alguns navegadores suportam o nome da cor.

  • atributo face:Seleciona a fonte indicada. O navegador assume que o usuário que acessa o site possui a fonte instalada. Caso não haja, o navegador usará a fonte padrão.

    Para garantir que as fontes estarão adequadas, podemos passar mais de uma fonte pelo parâmetro.
