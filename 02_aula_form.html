Formulários HTML

Formulários permitem que o usuário envie informações ao servidor através de campos de dados.
Cada campo possui um valor possível que pode ser preenchido, assim como atributos específicos que alteram o funcionamento do formulário.

Checkbox e Radio Button

Esses inputs permitem que o usuário específique informações de forma mais prática do que digitando as opções.
Cada um possui suas peculiaridades, que podem ser melhor explicadas com um exemplo. Aqui, iremos simular um pedido de pizza.

Você selecionou pizza de calabresa

Quais opções você deseja?

+ Queijo
+ Calabresa
+ Cebola
+ Azeitona

Checkbox permite que mais de uma opção seja selecionada. É caracterizado pelos botões quadrados.
Através do atributo "name", o backend identifica que os valores se referem à mesma sessão do formulário.
Contudo, neste caso, se os valores forem passados para "optional", sem colchetes, apenas o valor do último checkbox será contabilizado.
Isso porque o backend entende que a variável optional é primitiva, ou seja, recebe apenas um valor.
Para que um checkbox receba múltiplos valores, como nesse caso, é necessário passar uma lista como variável.
Por isso, optional[] é passado com colchetes, assim permitindo que múltiplas opções sejam contabilizadas.
Também é possível desabilitar a opção mas deixá-la visível ao usuário com o atributo disabled, como demosntrado na opção + Queijo.

Radio permite apenas que um valor seja selecionado no escopo. Ou seja, trabalha geralmentes com valores booleanos ou opções com apenas uma escolha.
Através do atributo "name" a página e o backend identificam que as opções pertencem à mesma sessão do formulário, permitindo que apenas um dos botões fique ativo.

Tipos de Buttons

Botões geralmente tem suas aplicações ampliadas e específicadas através do Javascript. Porém, o HTML permite algumas opções por padrão:

type="button"

Botão comum. Não interaje de nenhuma forma até ter um evento vinculado quando clicado. Esse evento pode ser designado pelo atributo "onclick" ou diretamente via javascript.

type="reset"


Botão que limpa o formulário ao qual está vinculado.

type="submit"

Botão que envia as informações do formulário ao qual está vinculado.

Tag Select

Esta tag restringe as opções que o usuário pode selecionar, permitindo assim que o formulário sempre receba as respostas esperadas, sem a possibilidade de o usuário inserir uma resposta indesejada.

Tag Textarea

Recebe uma string sem restrições. A área para digitação é maior, permitindo assim que o usuário digite textos mais longos.
O atributo 'rows' indica quantas linhas o input receberá, aumentando assim sua área de digitação.
